dateTimePicker

  • 需要修改样式，请用样式覆盖，或者修改项目源文件，然后执行 gulp build 命令重新生成文件
  • 样式文件已经整合到打包文件中了，使用前只需将 js 文件直接引入到页面中即可（请确保 jquery 已经在插件前面引用）
  • 因为使用了 webpack 打包，对于 IE7 和 IE8，需要引入 es5-shim.min.js 使其支持 es5 语法拓展，插件才能正常运行
